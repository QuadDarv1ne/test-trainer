# Тренажёр тестирования — Генератор тест-кейсов

Интерактивный веб-тренажёр для студентов по генерации тест-кейсов методами «чёрного ящика».

## Возможности

- **6 заданий** разной сложности (Легко / Средне / Сложно):
  1. Факториал — базовые классы эквивалентности и граничные значения
  2. Простое число — нелинейные классы эквивалентности
  3. Калькулятор скидки — многофакторное тестирование
  4. Високосный год — логические условия
  5. Треугольник — комбинаторное тестирование
  6. Валидация пароля — проверка форматов

- **Тренажёр**: создание тест-кейсов с входами, ожидаемым результатом, категорией
- **Автоматическая проверка**: эталонные функции выполняются в браузере
- **Оценка**: покрытие EC (40%), граничных значений (30%), корректность (30%)
- **Результаты**: анимированные диаграммы, детальная таблица, рекомендации
- **Теория**: обучающий контент о методах тестирования
- **Тёмная/светлая тема** с автоопределением системных настроек
- **Сохранение прогресса** в localStorage
- **Экспорт результатов** в буфер обмена

## Технологии

- **Next.js 16** (App Router)
- **TypeScript 5**
- **Tailwind CSS 4** + **shadcn/ui**
- **Framer Motion** (анимации)
- **next-themes** (тёмная тема)
- **sonner** (уведомления)
- **Lucide React** (иконки)

## Установка и запуск

### Требования

- **Node.js** >= 18.17
- **npm**, **yarn** или **bun**

### Установка

```bash
# Распакуйте архив и перейдите в папку проекта
cd test-trainer

# Установите зависимости (один из вариантов):
npm install
# или
yarn install
# или
bun install
```

### Запуск

```bash
# Режим разработки:
npm run dev
# или
bun run dev

# Сборка для продакшена:
npm run build
npm run start
```

Приложение будет доступно по адресу: **http://localhost:3000**

## Как пользоваться

1. Выберите задание на вкладке **«Задания»**
2. Изучите описание функции, сигнатуру, классы эквивалентности и граничные значения
3. Добавляйте тест-кейсы на вкладке **«Тренажёр»**:
   - Введите входные значения
   - Укажите ожидаемый результат
   - Выберите категорию (нормальное / граничное / исключение / недопустимый тип)
4. Нажмите **«Проверить»** — система сравнит ожидания с реальными результатами
5. Изучите результаты на вкладке **«Результаты»** с рекомендациями
6. Прочитайте теорию на вкладке **«Теория»**
